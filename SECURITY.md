# Security Policy

## Supported Versions

Only the latest release of brainkit receives security fixes.

## Reporting a Vulnerability

Please do not open a public issue for security problems.

Report vulnerabilities privately via
[GitHub private vulnerability reporting](https://github.com/Paldom/brainkit/security/advisories/new).

You can expect an initial response within 7 days. Please include a description
of the issue, steps to reproduce, and the affected version or commit if known.
